# 个人自用emacs配置

## 功能
* 自动补全
* 美化
    * 使用Dracula theme
* 模块化配置
    * 键盘布局
    * 插件配置+镜像源
    * 编辑器行为配置
    * 针对scheme语言进行高效编程
        * 添加chez-scheme钩子，c-x c-e即可自动执行上下文对应的s表达式
        * 配置结果分栏展示，用于开发调试和学习
* 其他
    * TBD

## 使用
* 备份原有emacs配置
```
mv ~/.emacs ~/.emacs.bak
mv ~/.emacs.d ~/.emacs.d.bak
```
* 克隆仓库
```
git clone https://github.com/jokergaming/emacs-config.git

```
* 打开emacs稍等片刻等待下载插件即可
